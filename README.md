# 🎬 IMDb Top250 影迹

> 汇总所有曾进入 IMDb Top 250 的电影，并展示其历史排名轨迹。
>  A historical aggregation of every film that has ever appeared in IMDb Top 250.

## 📖 项目简介

IMDb Top 250 是全球影迷广泛参考的重要电影榜单，但它是一个**实时动态榜单**。
 电影会随着评分变化进出榜单，排名也每天波动。

因此，某一个时间点的 Top 250，并不能完整反映这些电影的历史表现。

本项目基于 **top250.info** 提供的每日历史榜单数据，抓取并汇总所有历史上曾进入 IMDb Top 250 的影片，构建了一份完整的历史片单数据库。

这不仅是一份 Top 250，更是一份 **IMDb 榜单的历史档案**。

------

## ✨ 项目特色

- 📊 汇总所有历史入榜影片（而非仅当前 Top 250）
- 📈 展示最高排名 / 最低排名 / 平均排名
- ⭐ 记录最高评分 / 最低评分 / 平均评分
- 🕒 统计在榜天数与在榜比例
- 📅 显示最近在榜日期
- 🔎 支持模糊搜索（中 / 英文名 / 年份）
- 🎛 多维排序与筛选
- ✅ 本地标记“已观看”
- 💾 支持本地 JSON 备份与恢复
- 🌓 纯前端实现，无数据库依赖

------

## 🗂 数据来源

- 历史榜单数据来源：**top250.info**
- 原始榜单来源：**IMDb Top 250**

> 本项目仅做数据汇总与展示，不提供 IMDb 原始数据接口。

------

## 🧠 数据说明

每部影片包含以下统计字段：

| 字段         | 说明                       |
| ------------ | -------------------------- |
| 年份         | 上映年份                   |
| 最高排名     | 在榜期间达到的最佳名次     |
| 最低排名     | 在榜期间最低名次           |
| 平均排名     | 在榜期间平均名次           |
| 最高评分     | 在榜期间最高评分           |
| 最低评分     | 在榜期间最低评分           |
| 平均评分     | 在榜期间平均评分           |
| 在榜天数     | 累计进入 Top250 的天数     |
| 在榜比例     | 占统计周期的比例           |
| 最后在榜日期 | 最近一次出现在榜单中的日期 |

## 📌 项目初衷

IMDb 的 Top 250 每天都在变化。
 有的电影短暂上榜，又悄然离开；
 有的电影长期坚守，成为时代标志。

我希望将这些“曾经进入 Top 250 的电影”完整保留下来，
 让榜单不只是一张当下的快照，而是一条可回溯的轨迹。

独乐乐不如众乐乐，于是开源分享。
